En el área de Excel, el aprendiz desarrolla habilidades para organizar, calcular y analizar datos de forma eficiente, aplicando herramientas fundamentales para el entorno laboral. Se inicia con el reconocimiento de la interfaz, manejo de celdas, hojas de cálculo y formato de datos. Se aprenden funciones básicas y avanzadas como SUMA, PROMEDIO, SI, BUSCARV, entre otras, que permiten automatizar cálculos y tomar decisiones a partir de la información. También se trabaja con herramientas de ordenamiento, filtros, validación de datos y formatos condicionales para facilitar el análisis..
