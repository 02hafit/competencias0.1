HTML

En HTML (HyperText Markup Language) el aprendiz adquiere conocimientos básicos para crear y estructurar páginas web. Aprende a usar etiquetas para definir elementos como textos, imágenes, enlaces, listas y tablas. También se explora la organización del contenido usando etiquetas como <"head">, <"body">, <"h1">, <"p">, <"a">, entre otras. HTML es la base del diseño web y permite desarrollar sitios visualmente ordenados y funcionales. Es una herramienta esencial para entender cómo funciona internet y la creación de contenidos digitales.
